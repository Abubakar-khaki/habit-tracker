# 🚀 Habit Tracker

A simple and modern daily habit tracking web app.

## ✨ Features

* Add and manage daily habits
* Track daily completion
* Automatic daily reset system
* Streak tracking system
* Clean modern UI (dark/light mode)
* Data saved in browser (LocalStorage)

## 🌐 Live Demo

[👉 Start - Click Here](https://abubakar-khaki.github.io/habit-tracker/habit-tracker.html)

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## 📌 How it Works

* Each day is tracked separately
* Habits reset daily
* Progress is saved locally
* Streaks are calculated automatically

## 📁 Project Structure

* index.html → UI
* style (inside HTML)
* script (inside HTML)

## 🎯 Purpose

This project is built for learning and portfolio demonstration.
